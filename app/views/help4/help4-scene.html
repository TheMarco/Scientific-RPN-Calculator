Pi and e are well known constants. Phi is the so called 'divine proportion' and finally 42 is the most important constant in the universe. All four constants are stack lifting. It's therefore not needed to press after using one of these constants.

Number Altering Functions

Number altering functions modify the value in the X register.

Integer Portion replaces the number in the display with the nearest integer value.

Fractional Portion replaces the number with its fractional part.

Rounding rounds the number in the display to whatever format has been configured with FIX, SCI or ENG.

Absolute value produces the absolute value of the number in the display.

Trigonometric Functions

The trigonometric functions work in the selected mode. Selecting a mode tells the calculator what unit to use when performing trigonometric operations. The calculator supports three modes: degrees, radians and grads. Note that degrees are in decimals, NOT in minutes-seconds form.

The continuous memory will retain the selected trigonometric mode inbetween sessions.

Two-Number Functions

Two-number math is always performed using two sequentially entered values. If you're keying them in, make sure you use to separate them unless you have been using another function that terminates digit entry. For all two-number functions, the value entered first is considered the y-value. The second entered value is used as the x-value. In other words: The X and Y registers of the stack are used to perform the operation. As mentioned before, , , and are the four basic two-number functions. But there are more!

Percentual Difference

The percentual difference function which can be used by pressing calculates the percentual difference between two numbers. The result can be both positive and negative depending on which of the two entered numbers is the highest. This result represents a percentual increase or decrease.

Polar and Rectangular Coordinate Conversions

Pressing converts a set of rectangular coordinates (x, y) to polar coordinates. The y value must be entered first, then enter and finally the x value.
