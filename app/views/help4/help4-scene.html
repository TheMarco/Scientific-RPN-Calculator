The majority of functions on this calculator are numeric functions. All numeric functions automatically terminate digit entry. This means does NOT need to be pressed after using a numeric function.

Constants

There are four constants available: Pi, e, phi and 42. They are accessible as follows:

Pi and e are well known constants. Phi is the so called 'divine proportion' and finally 42 is the most important constant in the universe. All four constants are stack lifting. It's therefore not needed to press after using one of these constants.

Number Altering Functions

Number altering functions modify the value in the X register.

Integer Portion replaces the number in the display with the nearest integer value.

Fractional Portion replaces the number with its fractional part.

Rounding rounds the number in the display to whatever format has been configured with FIX, SCI or ENG.

Absolute value produces the absolute value of the number in the display.
